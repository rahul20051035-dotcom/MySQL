-- What is SQL?

-- SQL (Structured Query Language) is a standard programming language used to store, manage, retrieve, and manipulate data in a relational database.

-- SQL helps us talk to the database.

-- If a database is like a big digital cupboard 📂, SQL is the language we use to open it, search data, insert data, update data, and delete data.

-- Why Do We Use SQL?
-- We use SQL to:
-- Insert data into tables
-- Retrieve data (SELECT data)
-- Update existing data
-- Delete data
-- Create and modify tables & databases
-- Control access & permissions

-- Uses of SQL (Real-World)
-- SQL is used in:
-- 🏦 Banking systems (account details, transactions)
-- 🛒 E-commerce websites (orders, customers, products)
-- 🏥 Hospitals (patient records)
-- 🎓 Colleges (student management systems)
-- 📊 Data analysis & Business Intelligence
